# Showcase: Telehealth Bot (Regulated HealthTech)

This showcase demonstrates how CouncilAI handles **High-Stakes, Regulated** environments by enforcing HIPAA compliance and medical safety gates.

## 🚀 The Input
**The Idea:** "A HIPAA-compliant medical triage chatbot for remote areas to help prioritize patients for tele-consultations."
**Command:** `council init "A HIPAA-compliant medical triage chatbot for remote areas" --project telehealth-bot`

---

## 🛠️ The Process (Automated)
**Command:** `council all --project telehealth-bot`

The system identifies medical risks early and ensures that the technical architecture (data residency, encryption) aligns with the regulatory control register.
