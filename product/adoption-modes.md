# CouncilAI Adoption Modes

## Mode 1: Protocol-Only
- Teams manually follow agent prompts
- Human-led execution
- Ideal for early adopters

## Mode 2: AI-Assisted
- LLMs execute agent prompts
- Artifacts committed to Git
- Council governs outcomes

## Mode 3: Automated (Planned)
- Runner executes agents
- CI/CD enforces rules
- Dashboards provide visibility

Progression between modes is intentional and reversible.
