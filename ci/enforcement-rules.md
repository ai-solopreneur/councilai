# CouncilAI CI/CD Enforcement Rules

## Pull Request Blocking Conditions
A PR MUST be blocked if:
- Required agent artifacts are missing
- Council decisions are missing for high-risk changes
- Compliance matrix is outdated
- Tests are not updated
- Risk register is not updated

---

## Approval Rules
- High-risk changes require council approval
- Compliance overrides must be documented
- Emergency changes require post-hoc council review
