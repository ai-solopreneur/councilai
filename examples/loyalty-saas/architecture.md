# Architecture – Loyalty Program SaaS

## Stack
- Frontend: React + Tailwind
- Backend: Python FastAPI (OpenAPI)
- Database: DynamoDB
- Auth: AWS Cognito
- Infra: AWS Serverless

## Key Components
- API Gateway
- Lambda microservices
- Event-driven rewards engine
- Audit logging

## Scalability
- Horizontal scaling via Lambda
- Partitioned DynamoDB tables
